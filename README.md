# 🌦️ Real-Time Weather Dashboard

## Project Description
The Real-Time Weather Dashboard is a web application that fetches and displays live weather information for any city using a public REST API. This project demonstrates asynchronous JavaScript concepts, API integration, JSON data processing, and dynamic DOM manipulation to create an interactive user experience.

## Key Features
- Fetch real-time weather data using the Fetch API
- Utilize modern JavaScript async/await syntax
- Search weather information by city name
- Display temperature, humidity, wind speed, and weather conditions
- Parse and render nested JSON data dynamically
- Handle network errors and invalid city searches gracefully
- Responsive and user-friendly interface

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- RESTful Weather API
- JSON

## Learning Objectives

## How It Works
1. The user enters a city name in the search field.
2. A request is sent to the weather API using the Fetch API.
3. The API returns weather data in JSON format.
4. JavaScript processes the response and extracts relevant weather information.
5. The dashboard dynamically updates to display the weather details.

A dynamic and interactive weather dashboard that provides accurate, real-time weather metrics such as temperature, humidity, and wind speed based on user input.
